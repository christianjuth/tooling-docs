# Issues
| Command                  | What it does           |
| ------------------------ | ---------------------- |
| gh issue list -S "query" | Search issues          |
| gh issue list -l label   | Search issues by label |

# PR
| Command               | What it does        |
| --------------------- | ------------------- |
| gh pr list -A @me     | Search for my PRs   |
| gh pr list -S "query" | Search PRs by title |
| gh pr list -l label   | Search PRs by label |
